COMPILATION
=============================================

Activer l'environnement XXXXXX
conda activate XXXXXX

Aller au répertoire
cd path/to/folder

Lancer la compilation
pyinstaller XXXXXX.spec




DOCUMENTATION
=============================================

Créer un dossier documentation (par Windows)

Activer l'environnement XXXXXX
conda activate XXXXXX

Aller au répertoire
cd path/to/folder/documentation

Lancer le quick start de sphinx
sphinx-quickstart
(répondre y pour séparer sources et build)

Créer les fichiers sources automatiquement
sphinx-apidoc options -o path/to/doc/source path/to/code/sources

Options recommandé
--force pour réécrire tout
--separate pour créer un fichier par module

si besoin de refaire le TOC
--full réécrit tous les fichiers, dont conf (à sauver donc avant) mais surtout index, donc refait le toc

Au final, si on est toujours dans le dossier /documentation
sphinx-apidoc --force --separate --full -o source ../sources
(ou sans mise à jour du TOC)
sphinx-apidoc --force --separate -o source ../sources

Modifier le .conf (s'inspirer d'un existant...)

Générer la doc
make html
